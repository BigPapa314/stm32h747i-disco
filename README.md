stm32h747i-disco
================

_stm32h747i-disco_ contains a basic board support package for the
[STM32H747I-DISCO][] microcontroller board to write firmwares using the Rust programming language. 

Supported Components
--------------------

- [ ] LEDs
- [ ] Display
- [ ] Ethernet
